HistoryRun

HistoryRun was my final project for college. We were given the prompt of finding a problem to solve, and HistoryRun ended up becoming the future inspiration for LevelUpLearning. I had done a bunch of research on using games as a tool for teaching, and I ended up creating a prototype for a game that would help teach middle school students American history via a video game. The objective was to create a game that teach kids via experience.

A published interactive version of the project: HistoryRun

The First Steps

One of the best methods of learning is through personal experience. When someone remembers things based off of visual, olfactory, or auditory stimuli, they are less likely to forget that memory. Using this knowledge, I wanted to put the gamer in the middle of historic events, and have them experience those events first hand.

The Challenges

The task of representing a game without having any game experience is probably the most challenging thing I ever had to work around. My main focus was on the UI aspects of the project, specifically how the player would interact with the world, and how to design icons, menus and informative containers.
